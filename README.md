# aula-live-backend-realtime
Servicios WebSockets + WebRTC Proyecto AulaLive - Proyectio Integrador Univalle

## Documentacion Realtime (AsyncAPI)

- Especificacion fuente: [public/docs/asyncapi.yaml](public/docs/asyncapi.yaml)
- URL local del YAML: http://localhost:4000/docs/asyncapi.yaml
- URL local de visualizacion: http://localhost:4000/docs

En Render:

- URL YAML: https://<tu-servicio>.onrender.com/docs/asyncapi.yaml
- URL visualizacion: https://<tu-servicio>.onrender.com/docs

La ruta `/docs` redirige automaticamente a AsyncAPI Studio usando la URL publica del YAML hospedado en este servicio.
